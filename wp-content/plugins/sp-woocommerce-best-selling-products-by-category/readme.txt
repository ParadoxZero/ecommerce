A quick, easy way to add an best selling products by category to woocommerce.

== Description ==

This is very simple plugin to display woocommerce best selling products by category

View [DEMO](http://demo.wponlinesupport.com/shop/wc-best-selling-products-by-category/) for additional information.

Check [Best Selling Product Slider with category](https://wordpress.org/plugins/woo-product-slider-and-carousel-with-category/) for additional information.

View [PRO DEMO](http://wponlinesupport.com/wp-plugin/woo-product-slider-carousel-category/) for more details.

Shortcode example : 
<code>[bestselling_product_categories cats="CATEGORY_ID" per_cat="6" columns="3"] </code>


With this plugin you can now display woocommerce best selling products by category.

== Installation ==

1. Upload the 'sp-woocommerce-best-selling-products-by-category' folder to the '/wp-content/plugins/' directory.
2. Activate the sp-woocommerce-best-selling-products-by-category plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode <code>[bestselling_product_categories cats="CATEGORY_ID" per_cat="6" columns="3"] </code> to display the best selling products by category
